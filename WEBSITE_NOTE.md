# Note on the website/ subdirectory

The `website/` folder contains a presentation site that goes beyond the formal v1.00 submission. It is NOT part of the graded deliverable.

The website includes a section titled "Beyond the submission — personal follow-up experiments" which presents additional GPU-based experiments conducted AFTER the submission was tagged v1.00. These follow-up experiments use a different architectural configuration (bidirectional, multi-layer, seq-to-seq head) than the project's official architecture per PRD_models.md § 5.

If you are evaluating this project for grading purposes, please limit your review to:

- README.md (root)

- docs/ (all PRDs, ADRs, TODO, PROMPTS, audit, experiments)

- src/, tests/, config/

- pyproject.toml, uv.lock, .gitignore, .env-example

- assets/figures/ (FIG-1 through FIG-6)

The website/ directory is supplementary material for human readers and does not affect the formal submission.
